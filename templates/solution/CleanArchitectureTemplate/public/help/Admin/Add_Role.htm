Role

Admin can add a new role  by clicking a “New Role” button.

After clicking the button, the admin should enter the “Name,and function” ,then create the new role by clicking the “Save” button.

Admin can edit and delete Roles by clicking on the “three dots” in front of the name.

when user click on the edit button user can change a "Name ,Functions and Status" and save it using "Save" button.

when user click on the Delete button user can permanently delete the bank or cancel a deletion using the  buttons below.
