Licence Approval

After login this page users can see a licence approvals and users can add a new licence approval  by clicking a “New Licence Approval” button.

 

After click on button Admin should enter the  name and create a new licence approval using "Save" button.

 

User can edit licence approval details, and delete the bank by clicking on the “three dots” in front of the licence approval name.

 

when user click on the edit button user can change a "Name ,Status" and save it using "Save" button.

 

when user click on the Delete button user can permanently delete the bank or cancel a deletion using the  buttons below.
